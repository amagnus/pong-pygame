pong-pygame
===========

This is a simple implementation of pong using pygame.
Needs to run in desktop environment.

Run pong with:

`python run.py`

Requirements
===========

__Mac__

pygame 1.9.2pre
http://www.pygame.org/ftp/pygame-1.9.2pre-py2.7-macosx10.7.mpkg.zip

__Other platforms__

http://pygame.org/download.shtml
